# Split-String-By-Number
This simple script that will split any string by a number and returns a array.

## How to use
```javascript
//Number to split by
n = 3;
//Text to be split
t = "abcdefghijklm";
//Prints array that was retuned
console.log(split(t, n));
```

## What is being printed
```
[ 'abc', 'def', 'ghi', 'jkl', 'm' ]
```
